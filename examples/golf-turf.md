# Example: Golf Turf Inspection

## User Input

Scenario: golf turf  
Area: fairway section near sprinkler zone 4  
Turf: not sure  
Weather: hot afternoons, no rain for 8 days  
Irrigation: automatic sprinklers nightly  
Observation: irregular yellow-brown patches, worse on high-traffic area  

## Expected Agent Focus

- Compare affected patch with healthy nearby turf.
- Probe soil moisture in both areas.
- Check irrigation coverage and sprinkler overlap.
- Review mowing height and recent traffic.
- Look for circular patches, streaks, or localized dry spots.

## First Report Summary

Possible causes include localized dry spot, irrigation coverage issue, compaction, heat stress, mowing stress, or disease pressure. The agent should recommend soil moisture checks, irrigation inspection, and follow-up photos before any treatment.
