# Example: Home Garden Inspection

## User Input

Scenario: home garden  
Plant: tomato  
Location: raised bed  
Weather: warm and humid  
Watering: every morning  
Observation: yellow lower leaves and a few brown spots  

## Expected Agent Focus

- Ask whether the issue is spreading.
- Check watering and drainage.
- Inspect leaf underside.
- Compare old leaves and new growth.
- Suggest removing heavily damaged leaves only if appropriate.
- Recommend better airflow and careful watering habits.

## First Report Summary

Possible causes include overwatering, poor airflow, early disease pressure, nutrient stress, or normal lower-leaf aging. The agent should keep advice simple and ask for clearer photos before stronger conclusions.
