# Contributing

AI Farm Copilot is early. The most valuable contributions are practical workflows, realistic examples, field templates, safety improvements, and hardware integration notes.

## Good First Contributions

- Add an inspection example from a vineyard, orchard, golf course, garden, or hobby farm.
- Improve the skill intake questions.
- Add a CSV template for a new sensor or field log.
- Add a hardware note for a low-cost camera, drone, or sensor.
- Improve safety language for local expert review.
- Translate examples into another language.

## Example Contribution Format

For a new scenario example, include:

- Scenario.
- Crop, turf, or plant type.
- Weather.
- Irrigation or watering.
- Observation.
- Photos if available.
- Expected report summary.
- Any safety concerns.

## Safety Rule

Do not submit content that presents AI output as a confirmed diagnosis or final chemical recommendation. The project is for scouting support, field notes, and decision preparation.

## Development

The current demo is a static HTML file. Open `demo/index.html` in a browser.

No build step is required for V0.
