AI agent skill + local demo for turning plant, turf, weather, soil, and sensor observations into farm and garden inspection reports.

## LinkedIn Launch Draft

I am starting an open-source project called AI Farm Copilot.

The idea is simple: many farm, garden, vineyard, orchard, and golf turf monitoring systems are expensive and closed. But a lot of useful decision support can start with something much lighter:

- a phone photo
- field notes
- weather
- irrigation history
- soil or sensor readings
- an AI agent with the right workflow

AI Farm Copilot packages those workflows into a reusable open-source skill for Codex, Claude, and other agents. The first version supports vineyards, orchards, golf turf, hobby farms, and home gardens.

The long-term goal is to make this a free open decision layer that can connect to cameras, drones, sensors, mowing robots, spray robots, tractors, and other field equipment.

The core project will stay open-source. Commercial layers can come later through hosted dashboards, hardware kits, deployment support, and equipment integrations.

If you work in agriculture, turf, robotics, drones, sensors, or AI agents, I would love early feedback.

## Boss Update Draft

I started the first version of the AI Farm Copilot open-source repo. The first release is not trying to be a complete precision agriculture platform. It is a lightweight agent-native entry point:

- GitHub-ready README
- Codex/Claude skill package
- vineyard, orchard, golf turf, and home garden examples
- local demo
- CSV field and sensor templates
- safety disclaimer
- hardware integration notes
- 30-day launch plan
- investor brief

The goal is to publish something credible quickly, collect GitHub stars and pilot users, then use real usage to decide whether to expand into hardware kits, hosted dashboards, or partner integrations.
